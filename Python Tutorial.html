Introduzione a Python

Python è un linguaggio di programmazione versatile e potente, utilizzato per lo sviluppo web, l'analisi dei dati, l'intelligenza artificiale e molto altro.

Concetti Base di Python

  • Variabili e Tipi di Dati
  • Condizioni e Cicli
  • Funzioni
  • Liste e Dizionari
  • Gestione dei File
  • Eccezioni

Esempi Pratici

1. Variabili e Tipi di Dati

# Dichiarazione di variabili
nome = "Alice"
eta = 30
pi_greco = 3.14

# Stampa delle variabili
print(nome)
print(eta)
print(pi_greco)

2. Condizioni e Cicli

# Condizioni
numero = 10
if numero > 5:
    print("Il numero è maggiore di 5")
else:
    print("Il numero è 5 o minore")

# Cicli
for i in range(5):
    print(i)

3. Funzioni

def saluta(nome):
    return f"Ciao, {nome}!"

print(saluta("Mondo"))

4. Liste e Dizionari

# Liste
frutti = ["mela", "banana", "ciliegia"]
print(frutti[1])  # Output: banana

# Dizionari
studente = {
    "nome": "Alice",
    "eta": 20,
    "corso": "Informatica"
}
print(studente["corso"])  # Output: Informatica

5. Gestione dei File

# Scrittura su file
with open("esempio.txt", "w") as file:
    file.write("Ciao, Mondo!")

# Lettura da file
with open("esempio.txt", "r") as file:
    contenuto = file.read()
    print(contenuto)

6. Eccezioni

try:
    risultato = 10 / 0
except ZeroDivisionError:
    print("Errore: divisione per zero!")

7. Script Completo
