CSS

CSS (Cascading Style Sheets) è utilizzato per descrivere la presentazione di un documento HTML.

  • Home
  • Python
  • HTML

Introduzione a CSS

CSS consente di separare il contenuto dalla presentazione, rendendo le pagine web più belle e facili da mantenere.

Concetti Base di CSS

  • Selettori e Proprietà
  • Box Model
  • Flexbox
  • Grid Layout
  • Media Query
  • Transizioni e Animazioni
  • Trasformazioni

Esempi Pratici

1. Selettori e Proprietà

/* Selettori e Proprietà */
body {
font-family: Arial, sans-serif;
background-color: #f9f9f9;
}

h1 {
color: #333;
}

p {
color: #666;
line-height: 1.5;
}

2. Box Model

/* Box Model */
.box {
width: 200px;
padding: 20px;
border: 1px solid #000;
margin: 10px;
}

3. Flexbox

/* Flexbox */
.container {
display: flex;
justify-content: space-around;
align-items: center;
}

.item {
padding: 20px;
background-color: #ddd;
}

4. Grid Layout

/* Grid Layout */
.grid-container {
display: grid;
grid-template-columns: auto auto auto;
background-color: #f9f9f9;
padding: 10px;
}

.grid-item {
border: 1px solid #000;
padding: 20px;
text-align: center;
}

5. Media Query

/* Media Query */
@media (max-width: 600px) {
body {
    background-color: #ccc;
}
}

6. Transizioni e Animazioni

/* Transizioni e Animazioni */
.button {
background-color: #008CBA;
border: none;
color: white;
padding: 15px 32px;
text-align: center;
text-decoration: none;
display: inline-block;
font-size: 16px;
margin: 4px 2px;
transition-duration: 0.4s;
cursor: pointer;
}

.button:hover {
background-color: #555555;
color: white;
}

/* Animazione */
@keyframes example {
0% {transform: scale(1);}
50% {transform: scale(1.1);}
100% {transform: scale(1);}
}

.animate {
animation-name: example;
animation-duration: 4s;
}

7. Trasformazioni

/* Trasformazioni */
.box {
width: 200px;
height: 200px;
background-color: red;
transition: width 2s, height 2s, transform 2s;
}

.box:hover {
width: 300px;
height: 300px;
transform: rotate(180deg);
}
