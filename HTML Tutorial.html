Introduzione a HTML

HTML è il linguaggio di markup utilizzato per strutturare e presentare contenuti sul web.

Concetti Base di HTML

  • Struttura di base
  • Elementi e Attributi
  • Formattazione del Testo
  • Link e Immagini
  • Liste e Tabelle
  • Form e Input
  • Elementi Multimediali

Esempi Pratici

1. Struttura di base

<!DOCTYPE html>
<html lang="it">
<head>
    <meta charset="UTF-8">
    <title>Pagina di Esempio</title>
</head>
<body>
    <header>
        <h1>Benvenuti su CodeTutorial!</h1>
    </header>
    <footer>
        <p>© 2024 CodeTutorial. Tutti i diritti riservati.</p>
    </footer>
</body>
</html>

2. Elementi e Attributi

<p title="Questo è un paragrafo">Questo è un paragrafo con un attributo title.</p>

3. Formattazione del Testo

<h1>Questo è un titolo</h1>
<p><strong>Questo testo è in grassetto.</strong></p>
<p><em>Questo testo è in corsivo.</em></p>
<p><u>Questo testo è sottolineato.</u></p>

4. Link e Immagini

<a href="https://www.example.com">Visita il nostro sito</a>
<img src="immagine.jpg" alt="Descrizione dell'immagine">

5. Liste e Tabelle

<!-- Lista non ordinata -->
<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>

<!-- Tabella -->
<table border="1">
    <tr>
        <th>Nome</th>
        <th>Età</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>30</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>25</td>
    </tr>
</table>

6. Form e Input

<form>
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome">
    <br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    <br>
    <input type="submit" value="Invia">
</form>

7. Elementi Multimediali

<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    Il tuo browser non supporta il tag video.
</video>

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Il tuo browser non supporta il tag audio.
</audio>

8. Pagina Completa

<!DOCTYPE html>
<html lang="it">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Pagina Completa</title>
</head>
<body>
    <header>
        <h1>Benvenuti su CodeTutorial!</h1>
    </header>
    <nav>
        <ul>
            <li><a href="index.html">Home</a></li>
            <li><a href="python.html">Python</a></li>
            <li><a href="css.html">CSS</a></li>
            <li><a href="javascript.html">JavaScript</a></li>
        </ul>
    </nav>
    <main>
        <section>
            <h2>Introduzione a HTML</h2>
            <p>HTML è il linguaggio di markup utilizzato per strutturare e presentare contenuti sul web.</p>
        </section>
        <section>
            <h2>Elementi HTML</h2>
            <p>Gli elementi HTML sono i componenti fondamentali di una pagina web.</p>
            <ul>
                <li><strong>Paragrafo:</strong> <p>Testo di esempio</p></li>
                <li><strong>Link:</strong> <a href="https://www.example.com">Clicca qui</a></li>
                <li><strong>Immagine:</strong> <img src="immagine.jpg" alt="Esempio di immagine"></li>
                <li><strong>Lista non ordinata:</strong> 
                    <ul>
                        <li>Elemento 1</li>
                        <li>Elemento 2</li>
                        <li>Elemento 3</li>
                    </ul>
                </li>
                <li><strong>Tabella:</strong>
                    <table border="1">
                        <tr>
                            <th>Nome</th>
                            <th>Età</th>
                        </tr>
                        <tr>
                            <td>Alice</td>
                            <td>30</td>
                        </tr>
                        <tr>
                            <td>Bob</td>
                            <td>25</td>
                        </tr>
                    </table>
                </li>
            </ul>
        </section>
        <section>
            <h2>Form e Input</h2>
            <form>
                <label for="nome">Nome:</label>
                <input type="text" id="nome" name="nome">
                <br>
                <label for="email">Email:</label>
                <input type="email" id="email" name="email">
                <br>
                <input type="submit" value="Invia">
            </form>
        </section>
        <section>
            <h2>Elementi Multimediali</h2>
            <video width="320" height="240" controls>
                <source src="video.mp4" type="video/mp4">
                Il tuo browser non supporta il tag video.
            </video>

            <audio controls>
                <source src="audio.mp3" type="audio/mpeg">
                Il tuo browser non supporta il tag audio.
            </audio>
        </section>
    </main>
    <footer>
        <p>© 2024 CodeTutorial. Tutti i diritti riservati.</p>
    </footer>
</body>
</html>
